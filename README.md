# Простой список задач ToDo. Пет-проект.

Имеется: 
    
    кнопка добавления задачи.
    кнопка завершения задачи.
    кнопка удаления задачи.
    кнопка восстановления задачи.
    добавление даты и времени последних действий с задачью.
    переключение между задачами(активными, завершёнными, удалёнными).
    сохранение списка задач в localStorage.
    

Зависимости: 

    Нет зависимостей.
    

Запуск:
    
    1. npm i
    2. npm start
    3. npm run build-prod - для билд сборки (опционально)
